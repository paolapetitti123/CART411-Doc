    April 9th 2023 Update


    Between my last update to now a lot has happened including the laptop I was using to make this experience completely dying on me and having to buy a new computer. Nevertheless that did not stop this project at all, and in regards to coding I switched from doing this project in Vanilla JS to React and React Three Fiber. As of today I am declaring this version of my game to be finished. The game itself inside the TV screen is an iframe linking to my personal github page, as of now that portion of the game was coded entirely in p5JS.

    My Main goal for this project was to make something involving various skills I learned throughout my time in Computation Arts ranging from 3D modeling, to the several courses utilizing p5.JS, to photo & video editing. I also wanted to challenge myself to learn Three.js as this is something I've been wanting to learn for quite some time now.
